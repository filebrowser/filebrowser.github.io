Quick Start Installation Configuration

Features

Create, edit, copy, move, download your files easily, everywhere, every time. Use File Browser as your personal cloud.

Search through your files and find the one instantly. Filter by type, by extension...

Multiple users are supported, each one with their own folder, permissions and access roles.

Available in multiple languages! It's not available in yours? You can contribute and translate it yourself for the community.

Execute commands such as git, svn, hg, gofmt... on the web interface. They are configured by you so you can run whatever you want.

File Browser is a single executable file with no dependencies. Just download it and add it to your PATH. See Installation.

File Browser is available as a plugin for Caddy (a HTTP/2 web server with automatic HTTPS).

File Browser can be easily installed using Docker with docker pull hacdias/filebrowser.

File Browser plays well with static website generators, such as Jekyll and Hugo.
